# 📚 Книжный интернет-магазин

**Дипломный проект** — полнофункциональный веб-сайт для продажи книг с возможностью ознакомления с биографиями авторов и демо-версиями произведений.

---

## 🚀 Функционал

- 🔐 **Авторизация и регистрация** с функцией «Запомнить меня» (Flask-Login, сессии/cookies)
- 📖 **Каталог книг** с обложками, названиями и описаниями
- ✍️ **Биография авторов** для каждой книги (парсинг Wikipedia)
- 🔍 **Поиск по названию книги**
- 📄 **Демо-версия** каждой книги (ознакомительный фрагмент)
- 📞 **Контакты** (адрес, телефон, email, карта Екатеринбурга)

---

## 🛠 Технологии

| Компонент | Технология |
|-----------|------------|
| **Backend** | Python (Flask, Flask-Login, Flask-RESTful, Flask-WTF, SQLAlchemy) |
| **База данных** | SQLite / PostgreSQL (SQLAlchemy ORM) |
| **Парсинг** | BeautifulSoup4, requests (биографии авторов из Wikipedia) |
| **Фронтенд** | HTML, CSS, Bootstrap, Jinja2 |
| **API** | RESTful (JSON) |

---

## 📊 Структура БД (основные таблицы)

| Таблица | Описание |
|---------|----------|
| `users` | Пользователи (логин, email, хеш пароля) |
| `authors` | Авторы (имя, фото, ссылка на Wikipedia) |
| `books` | Книги (название, описание, иконка, связь с автором) |
